Conceptos de programación:
En la programación están las variables el cual es como una caja donde más adelante lo usaremos para guardar algo, la variable debe tener un nombre, el cual no debe ser muy corta, ni tan larga, debe ser una palabra que pueda describir de manera que se entienda que es lo que se encuentra en él, por ejemplo si la variable lo voy a usar para guardar la edad de algún usuario esto lo debe nombrar como (edad), para agregar una variable en JavaScript se pude usar var o let seguidamente de un espacio y el nombre de la variable después el signo igual, si el dato se ingresara más adelante se le coloca un 0 si este será un número, si es letra (string) se coloca (“ “).
Tipos de datos:
Aquí encontramos los números el cual los lenguajes de programación los reconocen, aunque estos sean naturales o decimales, la cadena de letras se conocen como string, por medio de los número podemos en los programas realizar operaciones matemáticas como la suma, resta, multiplicación, división, potencia, etc., los programas los reconocen como ( suma +, resta -, multiplicación *, división / y potencia como **) además de esto en los lenguajes podemos usar operadores comparativas ( igual ==, igual o menor que =<, igual o mayor que =>, menor que <, mayor que >, etc.,) y operadores lógicas que son los que tienen como resultado verdadero o falso ( y &&, 0 ||, negación !) estos los usamos en las condiciones en los programas, y normalmente su uso también se da en las matemáticas tanto como en nuestro día a día ya que siempre razonamos.
Estructuras de Control de Condiciones:
Para condiciones en nuestro programa usamos if el cual tiene la función de ejecutar el código si y solo si esta condición es verdadera, esta la if else, el cual que además de ejecutar el código si este es verdadero también tiene otra función si este fuera falsa y por último la if else, if else, este si es falsa tiene otra condición si este nuevamente diera falsa podría seguir una cadena de condiciones hasta que una resulte verdadera por ejemplo si una persona que usa el transporte público desea ir en un asiento, esperará en la parada un bus y verificara si este viene vacío o lleno, si este está lleno seguirá esperando sin importar si pasan varios hasta que venga un bus vacío para que pueda ir en un asiento, por último esta la swichtcase tiene una cantidad de alertas el cual le pertenece a distintos valores o respuestas, el cual dependiendo cual sea el valor o respuesta ingresada en la variable este mostrara la alerta que a este le pertenece.
Estructura de control de bucles:
El control de bucles nos sirve para repetir una cierta parte del programa una cantidad de veces dependiendo la cantidad de veces que deseamos o según la condición que se tenga, el ejemplo de bucle en JavaScript es for donde nosotros en la línea de código le indicamos la cantidad de veces máxima que este se pueda repetir, un ejemplo si queremos que 10 usuarios ingresen su nombre para un rifa, cada vez que alguien ingrese su nombre este sumara 1 a las veces que se ha mostrado, seguirá mostrándose hasta que la cantidad de veces sea igual a 10, este ya no se ejecutara porque ya tendría los nombres suficientes para la rifa, el while funciona dependiendo de las condiciones, podría hasta ser infinito, un ejemplo de esto son las paginas de tiendas en línea cuando agregamos algo al carrito esto nos pide si esto es lo único que compraremos o deseamos ver y agregar otro producto antes de realizar el pago.
Un ejemplo de bucle que vemos cada día es el semáforo.
